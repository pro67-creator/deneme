İşte **JavaScript** ile çözebileceğiniz çok **basit** 40 algoritma sorusu:

1. İki sayıyı topla.  
// Bu soru toplama işlemini ve fonksiyonlara parametre geçişini anlamanızı ölçer. İki sayının nasıl alınacağı ve toplama operatörünün (+) nasıl kullanılacağı öğrenilir.
2. Bir sayının karesini bul.  
// Bir sayının kendisiyle çarpılması işlemini uygular. Matematiksel işlemler ve fonksiyon kullanımı kavranır.
3. Bir sayının çift mi yoksa tek mi olduğunu yaz.  
// Modülüs (%) operatörüyle bir sayının 2’ye bölümünden kalana bakılır. Koşullu ifadeler (if-else) kullanılır.
4. 1’den 10’a kadar olan sayıları yazdır.  
// Döngü (for veya while) kullanımı ve artan sayılarla işlem yapma pratiği sağlar.
5. 10’dan 1’e kadar geriye doğru say.  
// Azalan döngü kurma ve sayaç değişkenini azaltma işlemi öğrenilir.
6. Dizideki ilk elemanı yazdır.  
// Dizi indekslemesi ve ilk elemana erişim ([0] ile) kavranır.
7. Dizideki son elemanı yazdır.  
// Dizi uzunluğunu bulup son elemana erişim ([dizi.length-1]) pratiği yapılır.
8. Dizideki eleman sayısını bul.  
// Dizi uzunluğunu (length) kullanarak eleman sayısı öğrenilir.
9. Dizideki tüm elemanları topla.  
// Döngüyle veya reduce fonksiyonuyla dizi elemanlarını toplama işlemi yapılır.
10. Dizideki en büyük sayıyı bul.  
// Döngüyle karşılaştırma veya Math.max ve spread operatörüyle en büyük değeri bulma yöntemleri öğrenilir.
11. Dizideki en küçük sayıyı bul.  
// Math.min veya döngüyle karşılaştırma yaparak en küçük değeri bulma pratiği yapılır.
12. Bir string’in uzunluğunu bul.  
// String’in length özelliğiyle karakter sayısı öğrenilir.
13. Bir string’i büyük harfe çevir.  
// toUpperCase() fonksiyonu ile string’in tüm harfleri büyük yapılır.
14. Bir string’i küçük harfe çevir.  
// toLowerCase() fonksiyonu ile string’in tüm harfleri küçük yapılır.
15. Bir string’in ilk harfini yazdır.  
// String’in ilk karakterine ([0]) erişim sağlanır.
16. Bir string’in son harfini yazdır.  
// String’in son karakterine ([string.length-1]) erişim sağlanır.
17. “Merhaba” kelimesini 3 kere yazdır.  
// Döngüyle aynı metni birden fazla kez ekrana yazdırma pratiği yapılır.
18. 5’in çarpım tablosunu yazdır (5×1 … 5×10).  
// Döngüyle 5’in farklı sayılarla çarpımı ve sonuçların ekrana yazdırılması sağlanır.
19. 0-100 arasında rasgele bir sayı üret.  
// Math.random() ve Math.floor() fonksiyonlarıyla rastgele sayı üretme öğrenilir.
20. Bir sayının faktöriyelini bul (örn. 5! = 120).  
// Döngü veya özyineleme (recursion) ile faktöriyel hesaplama pratiği yapılır.
21. Saat ve dakika parametrelerini alıp “12:05” formatında yaz.  
// String birleştirme ve sayıları iki basamaklı gösterme (padStart) yöntemleri öğrenilir.
22. Yaşını parametre olarak alıp “18 yaşındasın” yaz.  
// String birleştirme ve parametre kullanımı pratiği yapılır.
23. İki string’i birleştir (örn. "Mer" + "haba").  
// String birleştirme (concat veya + operatörü) işlemi öğrenilir.
24. String’in içinde “a” harfi varsa true döndür.  
// includes() fonksiyonu veya döngüyle karakter arama işlemi yapılır.
25. Bir sayının pozitif mi negatif mi olduğunu yaz.  
// Koşullu ifadelerle (if-else) sayının işareti kontrol edilir.
26. Dizideki tüm string’leri büyük harfe çevir.  
// map() fonksiyonu ve toUpperCase() ile dizi elemanlarını dönüştürme pratiği yapılır.
27. Diziden çift sayıları filtrele.  
// filter() fonksiyonu ve modülüs (%) operatörüyle çift sayılar seçilir.
28. Diziden tek sayıları filtrele.  
// filter() fonksiyonu ve modülüs (%) operatörüyle tek sayılar seçilir.
29. Ters çevrilmiş bir string döndür (örn. "abc" → "cba").  
// split, reverse ve join fonksiyonlarıyla string’i ters çevirme işlemi yapılır.
30. Bir sayının 0 olup olmadığını kontrol et.  
// Koşullu ifadelerle (if-else) sayının sıfır olup olmadığına bakılır.
31. Dizideki tekrar eden elemanları bul.  
// Döngü, filter veya Set kullanarak tekrar eden elemanlar tespit edilir.
32. İki sayıdan büyük olanı seç.  
// Koşullu ifadelerle (if-else) veya Math.max ile büyük olan bulunur.
33. İki sayıdan küçük olanı seç.  
// Koşullu ifadelerle (if-else) veya Math.min ile küçük olan bulunur.
34. 1’den 100’e kadar 5’e bölünebilen sayıları yazdır.  
// Döngüyle 1-100 arası sayıları kontrol edip 5’e tam bölünenleri yazdırma pratiği yapılır.
35. 1’den 100’e kadar 3’e ve 5’e bölünebilen sayıları yazdır.  
// Döngüyle 1-100 arası sayıları hem 3’e hem 5’e bölünebilenleri bulma işlemi yapılır.
36. String’in içindeki boşlukları sil.  
// replace veya split/join ile boşluk karakterlerini kaldırma işlemi yapılır.
37. String’in içindeki sesli harf sayısını bul.  
// Döngü veya filter ile sesli harfleri sayma işlemi yapılır.
38. Bir string’in palindrom olup olmadığını kontrol et (örn. "kayak").  
// String’in tersiyle kendisini karşılaştırarak palindrom kontrolü yapılır.
39. Dizideki elemanları tersten yazdır.  
// reverse fonksiyonu veya döngüyle diziyi tersten yazdırma işlemi yapılır.
40. Parametre olarak gün sayısını alıp bu gün hafta içi mi hafta sonu mu olduğunu yaz (0=Pazartesi…6=Pazar).  
// Koşullu ifadelerle (if-else veya switch-case) günün hafta içi mi sonu mu olduğu belirlenir.

41. Bir dizideki tüm elemanların ortalamasını bul.  
// Dizi elemanlarını toplayıp eleman sayısına bölerek ortalama hesaplanır. Matematiksel işlemler ve döngü kullanımı öğrenilir.
42. Bir string’in baş harfini büyük yap, diğer harflerini küçük bırak.  
// String işlemleri, substring ve toUpperCase/toLowerCase fonksiyonlarıyla karakter manipülasyonu yapılır.
43. Bir sayının asal olup olmadığını kontrol et.  
// Döngüyle 2’den başlayarak sayının bölenleri kontrol edilir. Koşullu ifadelerle asal sayı kontrolü yapılır.
44. Bir dizideki tüm elemanları tek bir string olarak birleştir (virgülle ayır).  
// join() fonksiyonu ile dizi elemanları araya virgül koyularak birleştirilir.
45. Bir string’in içinde kaç tane belirli bir harf geçtiğini bul.  
// Döngü veya split/filter ile belirli bir harfin sayısı hesaplanır.
46. Bir dizideki en uzun string’i bul.  
// Döngü veya reduce ile her string’in uzunluğu karşılaştırılır ve en uzun olan seçilir.
47. Bir sayının mutlak değerini bul.  
// Math.abs() fonksiyonu ile negatif sayılar pozitif yapılır, pozitifler değişmeden kalır.
48. Bir dizideki tüm elemanları rastgele sırala.  
// sort() fonksiyonu ve Math.random() ile dizinin sırası karıştırılır (shuffle işlemi).
49. Bir string’in sadece rakamlardan oluşup oluşmadığını kontrol et.  
// RegExp veya döngüyle karakterlerin hepsinin rakam olup olmadığı kontrol edilir.
50. Bir dizideki elemanların toplamı çift mi tek mi?  
// Dizi elemanları toplanır ve toplamın çift mi tek mi olduğu modülüs (%) operatörüyle kontrol edilir.
